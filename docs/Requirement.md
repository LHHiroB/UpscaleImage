# Requirement

## Objective

Nghiên cứu và xây dựng giải pháp AI Upscale Image có khả năng tăng độ phân giải ảnh với chất lượng cao và thời gian xử lý tối ưu.

## Functional Requirements

- Upscale ảnh từ độ phân giải thấp lên cao.
- Hỗ trợ nhiều model AI.
- Chạy hoàn toàn offline.
- Hỗ trợ CLI.

## Non-Functional Requirements

- Thời gian xử lý ≤ 90 giây/ảnh.
- Có thể benchmark nhiều model.
- Dễ dàng thay đổi model.
- Kiến trúc dễ mở rộng.

## Target

- Windows
- Python 3.11+
- GPU NVIDIA (ưu tiên)
